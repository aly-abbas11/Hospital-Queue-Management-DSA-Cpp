# Hospital Queue Management System — DSA C++ Project

A console-based hospital queue management system built in C++ 
that demonstrates core Data Structures and Algorithms concepts 
through a real world medical application.

## About
Built during my third semester as part of my Data Structures 
and Algorithms course. This project applies linked list and 
priority queue concepts to solve a real problem that hospitals 
face every day — managing patients efficiently based on urgency 
and arrival time.

## Features
- Add patients to Emergency and OPD departments
- Priority based queue ordering by department and patient type
- VIP and Elderly patients served before Normal patients
- Emergency department prioritized over OPD
- Arrival time used as tiebreaker for equal priorities
- Call next patient and remove from queue
- Display current queue for any department

## DSA Concepts Used
- Singly Linked List for queue implementation
- Priority Queue logic with custom comparator
- Dynamic memory allocation and deallocation
- Pointer based node insertion and deletion

## OOP Concepts Used
- Classes and objects
- Encapsulation of queue logic inside Department class
- Constructor initialization
- Separation of concerns between Case and Department classes

## Tech Stack
- Language: C++
- IDE: Dev-C++ / Code::Blocks

## How to Run
1. Clone the repository
2. Open the .cpp file in your IDE
3. Compile and run
